## servidor em cloud e On Premise:
### Gastos:
- um servidor em cloud nem sempre é mais barato em comparação a um servidor On Premise
	- A longo prazo o servidor On Premise é mais barato
	- O servidor em cloud é mais barato na implantação, mas pode se tornar mais caro ao longo do tempo

### Segurança:
-  A cloud muitas vezes não está alinhada com a LGPD

### Meio ambiente:
- cloud não é Footprint zero
	- ela gera sim insumos ambientais 
	- site para verificar a pegada ecológica: https://www.cloudcarbonfootprint.org/


## Ideias sobre o futuro da cloud

- próximo grande passo da evolução como o do homo Habilis

## projeto corrente no Brasil INCT InterSCity:
- projeto para desenvolver softwares para as cidades
- principal projeto do Brasil
- aproximadamente 100 pesquisadores e estudantes estão no projeto

  ### Foco principalmente em:
  -  pessoas(tecnologia é um meio, não um fim)
  - população de baixa renda
  - países em desenvolvimento
  - bairros menos favorecidos 


## projetos desenvolvidos pelo INCT InterSCity:

### 1. Plataforma InterScity (plataforma de sofware genérica para cidades inteligentes)
- uma plataforma que serve como base para se comunicar com todas as outras bases de dados, por exemplo, é uma placa mãe e os outros sistemas são os componentes dos computadores

### 2. Health Dashboard:
- Usou dados de internações do SUS para descobrir o quanto as pessoas tem que andar para poder ser atendidas pelo pelo SUS
- Além disso dá para descobrir onde está concentrado os casos de Leukemia, HIV, Dengue Fever e etc
- Verificar a área de abrangência de um hospital, desde onde as pessoas vem para serem atendidas por aquele hospital

### 3. Ranking the acessibilidade:
- é um site de avaliação de o quanto um lugar é acessível

### 4. Startup Scipopulis:
- aplicativo para verificar a posição do ônibus, além de ter outras features para tornar melhor a experiencia do usuário.

### 5. Mobilidade Ativa com BikeScience:
-  auxilia na análise de onde mais as pessoas que andam de bicicleta andam de bicicleta, para permitir uma análise de onde criar as ciclo vias.
