# Explicação dos tópicos:


## LGPD (lei geral de proteção de dados pessoais)
- inspirada na GDPR da união europeia
- cria ressalvas para proteger, resguardar e utilizar os dados dos usuários.


![[Pasted image 20241109083629.png]]

características da LGPD:
- Não transmita dados de sua empresa para outra sem a permissão do usuário

## etapas do processamento dos dados

![[Pasted image 20241109084621.png]]


existe uma validade para os dados, tipo compartilhei um dado com o banco e deixei sem fechar a conta, quanto tempo a empresa pode reter minhas informações?
Resposta: 5 anos após o encerramento da conta, mas ele pode manter os dados para sempre se não for encerrado.

## termo de consentimento
- ele vem depois que a lgpd
- mas se tiver uma lei de suporte vem primeiro que a lgpd

## Agentes de tratamento

![[Pasted image 20241109090743.png]]

ex:
- controlador: diretor, coordenador.
	- É quem controla o dado, quem faz as decisões sobre os dados
-  operador: técnico de TI, coletor de informações
	- quem coleta os dados, quem trata os dados
- encarregado: coordenador de curso(ex)
	- é quem se comunica entre os proprietários dos dados, controlador e a autoridade nacional de proteção de dados e quem verifica se as normas estão sendo cumpridas
	- com relação ao governo ele se comunica com a ANPD(Autoridade Nacional de Proteção de Dados)
	- deve ter suas informações de contato amplamente divulgados
