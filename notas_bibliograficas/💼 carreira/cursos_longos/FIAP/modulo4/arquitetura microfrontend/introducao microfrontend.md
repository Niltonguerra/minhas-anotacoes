## história do surgimento do micro frontend
### popularidade dos microservices
- surgi para deixar o backend mais escalavel
### limitações dos monólitos backend
- o micro serviços surgi para apenas como uma abordagem para o backend

### desafios com monólitos frontend
- nisso surgi uma necessidade de também escalar o frontend assim como o backend e nisso surgi o micro frontend


## problemas comuns com os monólitos
- dificuldade em integrar novas funcionalidades
- complicações com novas ferramentas de build
- escalabilidade e trabalho em equipe

## definição do microfrontend
- fragmentação em pedaços menores e independentes
- coesão para o usuário
