# Explicação dos tópicos:

## o que é o Style Guide?
- é basicamente um documento que tem todas as regras de design para uma marca 
	- tem coisas como:
		- cores
		- fontes
		- breakpoints
		- margens
		- e etc.
## características fundamentais
- todos os envolvidos no desenvolvimento do produto precisam ter acesso a ele

## benefícios:
- consistência
- colaboração facilitada
- Onboarding simplificado
- adaptar ao mercado
- melhoria contínua facilitada
- flexibilidade para a adaptação da marca
