## tipos de requisitos de sistema:
![[Pasted image 20250810155537.png]]
- requisitos funcionais
	- que precisa ter para que o sistema funcione
- requisitos não funcionais
	- aqueles que não são obrigatórios, mas são desejaveis
		- por exemplo:
			-  uma features que não agrega tanto valor ao usuário final
			- desempenho do código( muitas vezes mas nem sempre)
			- coisas pequenas que não fazem com que o valor principal seja negado
- requisitos de interface
	- layout
- requisitos de usabilidade
- requisitos de conformidade
- requisitos de manutenção e suporte
