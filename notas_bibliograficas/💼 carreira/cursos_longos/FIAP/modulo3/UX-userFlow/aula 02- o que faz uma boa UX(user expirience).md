- deve ser fácil e simples de usar
	- navegação clara
	- controles bem posicionados
	- feedback automático
## acessibilidade
- deve ser acessível
	- permitir que deficientes usem 
	- escolha das cores
	-  tamanho de fonte
## utilidade
- deve realmente ajudar o usuário

## desempenho
- deve ser livre de erros
	- ou pelo menos avisar sobre o problema
- o sistema deve ter disponibilidade

- deve ser agradável o uso dele
- o usuário deve se sentir confortável ao usar o aplicativo
