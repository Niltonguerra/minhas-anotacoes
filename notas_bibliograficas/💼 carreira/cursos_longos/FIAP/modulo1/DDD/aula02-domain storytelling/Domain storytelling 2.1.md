## Linguagem pictográfica
- usa desenho com palavras para representar o que está sendo transmitido para você para a criação do sistema
- nessa linguagem existe as seguintes coisas:
	- atores
	- objetos de trabalho 
	- atividades
	- números sequenciais 
	- anotações grupos
	- cores
![[Pasted image 20250414194124.png]]
- atores
	- o ator muitas vezes é uma entidade, alguém que realiza algo dentro do sistema
- objetos de trabalho 
	- são ferramentas utilizadas pelos atores para a realização de alguma atividade
- Atividade
	- são as ações que o ator realiza utilizando os objetos de trabalho
- números sequenciais
	- servem para indicar o fluxo da história, ou seja, o que vem primeiro
-  anotações
	- são anotações, kkkk
	- servem para anotar informações importantes sobre o sistema que se deve atentar, como, limitações, ações prévias, gatilhos que devem ser executados juntamente
- grupos
	- são processos que iram se repetir constantemente, subdomínios, limitações do processo e etc
- cores
	- utilizadas para dar ênfase para algumas partes do sistema

- identificar o dominio
- identificar o subdomínio principal
- identificar o domínio de suporte
- identificar o domínio de genérico
- juntar em uma sala todos os  domain experts em uma sala para discutir sobre o projeto e criar os diagramas
